# Primeros Pasos

Esta sección te guiará a través de la configuración inicial de **Nuevo Galeno**, para que puedas comenzar a trabajar de forma rápida y segura.

---

## Requisitos del sistema

Nuevo Galeno es una aplicación de escritorio diseñada para funcionar de manera fluida en equipos modernos.

**Requisitos mínimos recomendados:**

- Sistema operativo: Windows 10 o superior  
- Memoria RAM: 4 GB (8 GB recomendado)  
- Espacio en disco: 500 MB libres  
- Resolución mínima: 1366×768  

---

## Instalación

1. Ejecuta el instalador proporcionado.
2. Sigue los pasos del asistente de instalación.
3. Una vez finalizada la instalación, abre **Nuevo Galeno** desde el acceso directo.

---

## Primer inicio

Al iniciar la aplicación por primera vez:

- Se creará automáticamente la base de datos local.
- El sistema estará listo para comenzar a cargar información.
- No se requiere conexión a internet para su funcionamiento.

---

## Recomendaciones iniciales

Antes de comenzar a trabajar:

- Configura los datos del consultorio.
- Verifica la fecha y hora del sistema.
- Revisa la configuración general.
- Realiza una copia de seguridad inicial.

---

## Migración desde Galeno antiguo

Si utilizabas el sistema Galeno antiguo, Nuevo Galeno incluye un **importador especializado** que permite migrar:

- Pacientes  
- Tratamientos  
- Odontogramas  
- Pagos y saldos  
- Historia clínica  

Consulta la sección correspondiente para conocer el procedimiento paso a paso.

---

Con esto, tu sistema estará listo para comenzar a operar.
